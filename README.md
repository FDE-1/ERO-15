***PROJECT ERO***

This is a repository for the Project ERO made by the group PARIS15.

**Python libraryes**

    - NetworkX
    - OSMNX
    - threading

**Fichier d'execution**
    start.sh

**documentation**

*1. drone.py*
    - Donne le chemin et le prix du drone pour chaque district

*2. eulerize.py*
    - Transforme le graphe donnée en graphe eulerien

*3. district_map.py*
    - récupère la carte de l'arrondissement sous forme de graphe

*4. optimal_path.py*
    - Cherche le ou les chemins selon le nombre de véhicule
